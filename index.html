FrostByte

Save Gigabytes of RAM on Linux by auto-freezing idle applications.
Zero dependencies. Instant thaw. Built for GNOME & Wayland.

View on GitHub curl -sS https://raw.githubusercontent.com/VladislavTsytrikov/frostbyte/main/install.sh | bash

Auto-Suspend

FrostByte monitors CPU activity and sends SIGSTOP to apps that stay idle too long. RAM is reclaimed instantly by the kernel.

Lazy Thaw

Only the active process thaws immediately. Children (like Chrome tabs) wake up one-by-one to prevent system spikes.

Audio Aware

Never worry about your music stopping. FrostByte detects active audio streams and keeps players running.
