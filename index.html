Auto-Suspend

FrostByte monitors CPU activity via /proc and sends SIGSTOP to apps that stay idle too long. RAM is reclaimed instantly by the kernel.

Lazy Thaw

Only the active process thaws immediately. Children (like Chrome renderer tabs) wake up one-by-one to prevent system resource spikes.

Native GNOME

Includes a companion GNOME Shell extension for transparent Wayland support. It tracks focus and click events to wake apps instantly.

Technical Deep Dive

Most Linux "memory savers" rely on killing processes. FrostByte is different. It uses the kernel's own signaling mechanism to suspend processes without losing state.

  • Process Tree Awareness: Correcty handles complex multi-process applications like Chrome, Discord, and VS Code.
  • Audio Detection: Automatically whitelists any application currently playing audio via PulseAudio or PipeWire.
  • Zero Config: Intelligent defaults out of the box, with per-app rules available for power users.
  • Window-Aware: The daemon works in sync with the window manager to ensure an "instant on" experience.
